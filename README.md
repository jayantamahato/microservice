## Services

- Admin
- Customer
- Cart
- Order
- Product
- Shopping
- Vendor


## Tech Stack

**Server:** Node, Express, MongoDB, RabbitMQ, Cloudinary, AWS S3, Docker, K8s, TelegramBot, DiscordBot

## Run Locally

Clone the project

```bash
  git clone https://github.com/jayantamahato/microservice
```

Go to the project directory

```bash
  cd microservice
```
```bash
  cd proxy
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
